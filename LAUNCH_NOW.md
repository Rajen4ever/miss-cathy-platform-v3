# Launch Now

This pack gives you everything that can be completed without your private credentials.

## You can open now
- `public-site/index.html` in any browser
- `public-site/privacy.html`
- `public-site/terms.html`

## To make the website publicly accessible
1. Create a Supabase project
2. Add web env vars
3. Deploy `apps/web` to Vercel / hosting
4. Point a domain to the deployment
5. Add support email / legal entity details to privacy and terms pages

## To make the Android app downloadable from Play Store
1. Add mobile env vars
2. Build a signed Android App Bundle
3. Create Play Console app record
4. Upload signed AAB
5. Complete Data safety, content rating, and store listing
6. Run internal testing and then production rollout

## Honest boundary
No one can publish your website or Play Store app from this environment without your real accounts, credentials, signing keys, and console approvals.
