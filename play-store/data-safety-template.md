# Google Play Data safety template (draft)

Review and customize this in Play Console based on your final production configuration.

## Data collected
- Email address
- User-created content
- App activity / diagnostics (only if enabled)
- Push token (if notifications are enabled)

## Data purpose
- App functionality
- Account management
- Notifications
- Analytics / diagnostics (if enabled)

## Data handling notes
- Data is not sold
- Data deletion supported
- Data encryption in transit should be enabled
- Access should be restricted by authentication and RLS
